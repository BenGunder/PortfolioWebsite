Roblox Games


In the summer of this year, I decided that I wanted to make a game using an engine and language I had never used before. And so, I began a three month project of learning Lua and the Roblox Studio engine to create my game, Dungeons & Diners.

I decided to make this game in Roblox for a few reasons. The biggest of which is that Roblox makes it incredibly easy to build multiplayer games, which is something I had never made before. I had also enjoyed playing a lot of other Roblox games in the past, inspiring me to make my own, and I found that Roblox Studio is incredibly approachable to new users and had extensive documentation.

From working on this project, I learned a lot about programming multiplayer games with a client-server model. I also learned a lot about UI design and building 3D environments. Overall, I had a lot of fun working on this project, and I will definitely be interested in making another game in Roblox studio in the future.

My newest project, ESCAPE POD, is a survival horror game that I released the week before Halloween. Thanks to my experience with my previous game, I was able to complete this one a lot more efficiently, and I think it has a lot more polish than my last. I am really proud of what I was able to do with this project and had a ton of fun making it.
